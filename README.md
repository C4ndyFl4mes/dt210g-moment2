# DT210g Moment 2

## Beskrivning
Ett repo för en webbapplikation som kan hantera en todolista.

## Databas + API
API:et är ett .NET webapi i C# och hanterar en MySQL databas. Både databasen och API:et ligger på Railway.

## Frontend
Webbapplikationen är skriven i TypeScript och använder React som ramverk. Relevanta komponenter för uppgiften är TodosDisplay, CreateTodo och Todo. Det finns en statisk klass som håller i validering med hjälp av Yup.
